# ML Project

This is an end-to-end Machine Learning pipeline implementation.

## Project Structure

- `data/` - CSV data files
- `pictures/` - Visualization images
- `notebooks/` - Jupyter notebooks for EDA and analysis
- `ml_pipeline/` - Core ML pipeline implementation
- `visualization/` - Visualization functions
- `tests/` - Unit tests

## Setup

1. Install Poetry: `pip install poetry`
2. Install dependencies: `poetry install`
3. Activate virtual environment: `poetry shell`

## Usage

- Run notebooks: `jupyter notebook`
- Execute pipeline: `python -m ml_pipeline.main`
